# Issue Tracker

## Fejl og udviklingsønsker

Fejl og udviklingsønsker indberettes som issues på Github enten af kommunen selv eller af kommunens driftsleverandør.
Beskriv problemet eller forslaget så detaljeret som muligt.

Det kræver et login til Github for at kunne oprette issues. Man opretter sit eget login her: [https://github.com/signup](https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home)

[OS2Display Issue Tracker](https://github.com/os2display/display-api-service/issues){: .btn .btn-purple}


## Sagsbehandling
Alle sager behandles af OS2Display koordinationsgruppen. Koordinationsgruppen mødes en gang månedligt.
Hastesager, der vedrører sikkerhed eller driftsstabilitet, behandles hurtigere.

Spørgsmål til sagsbehandling kan rettes til [os2display@os2.eu](mailto:os2display@os2.eu)
